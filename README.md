# Mi Portafolio Profesional (mi-portafolio-final)

"Mi portafolio profesional desarrollado como parte del curso."

## Características Destacadas
* Diseño Responsive con Bootstrap 5.
* Cumplimiento de la regla 60-30-10 en la paleta de colores.
* Uso de HTML5, CSS3 (con animaciones y transiciones) y JavaScript.
* Formulario de contacto funcional (simulado) con validación.

## Estructura del Proyecto
El sitio web público se encuentra dentro de la carpeta `proyecto`.
